# geomet-data-registry Docker setup

Development environment using Docker

## Docker

```bash
git clone https://github.com/ECCC-MSC/geomet-data-registry.git
cd geomet-data-registry

# build image
make build

# run
make run

# login to a bash session
make login

# stop
make stop
```

## Docker Compose

```bash
docker-compose up -d
```
